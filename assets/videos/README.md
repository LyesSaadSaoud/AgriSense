# AgriSense mission videos

Place the three real video files in this folder using these exact names:

- `inspect_zone2.mp4`
- `inspect_allZones.mp4`
- `inspect_zone4.mp4`

The website automatically detects each file and replaces its “Coming soon” poster with the playable video.

The videos were not present in the supplied manuscript ZIP, so they are intentionally not fabricated or embedded here.
